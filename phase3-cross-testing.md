# Phase 3: Cross-Testing

**Duration:** ~25 minutes  
**Time:** 19:50-20:15  
**Goal:** Evaluate another team's AI feedback agent

---

## Step 1: Get Your Assignment (2 min)

- Facilitator will assign you to test **another team's repository**
- Note which team you're evaluating
- Navigate to their forked repository

---

## Step 2: Test Their Agent (10 min)

### Setup:
1. Open the assigned team's repository
2. Find their instruction file in `.github/instructions/`
3. Open **GitHub Copilot Chat**
4. Make sure Copilot is using their instruction file

### Test:
1. Paste **YOUR summary** from Phase 1
2. Ask: `"Give me feedback on this summary"`
3. Read the feedback carefully

### Observe:

Think about these 5 criteria while reading the feedback:
- **Understandability:** Could you comprehend what the agent was telling you?
- **Specificity:** Did the agent provide concrete examples vs. vague statements?
- **Validity:** Was the feedback accurate and aligned with the rubric?
- **Actionability:** Could you actually use this feedback to improve?
- **Affective Quality:** Did the feedback motivate you to improve?

---

## Step 3: Submit Feedback via GitHub Issue (13 min)

### Create Issue:
1. Go to the **main hackathon repository** (not the team's fork)
2. Click **Issues** tab
3. Click **New Issue**
4. Select **"Agent Feedback (Phase 3)"** template

### Fill Out the Form:

The form will ask you to:
1. **Identify groups** (your group and the group you tested)
2. **Rate on 5 criteria** using a 5-point scale (5=Excellent, 4=Good, 3=Adequate, 2=Needs Improvement, 1=Poor)
3. **Provide evidence** for each rating with specific examples

**Criterion 1: Understandability**  
*Could you comprehend what the agent was telling you?*
- **Rating:** Select from dropdown (5-Excellent to 1-Poor)
- **Evidence:** Provide a specific example or observation  
  *Example: "The agent clearly identified each issue and explained what needed to change, making it easy to understand exactly what was problematic."*

**Criterion 2: Specificity**  
*Did the agent provide concrete examples vs. vague statements?*
- **Rating:** Select from dropdown (5-Excellent to 1-Poor)
- **Evidence:** Provide a specific example or observation  
  *Example: The agent quoted my exact text "multiple minerals indicate" and explained specifically what was missing from that statement.*

**Criterion 3: Validity**  
*Was the feedback accurate and aligned with the rubric?*
- **Rating:** Select from dropdown (5-Excellent to 1-Poor)
- **Evidence:** Provide a specific example or observation  
  *Example: "The agent correctly identified that my Dimension 2 (Interpretation) was missing significance explanation, which aligned with the rubric criteria."*

**Criterion 4: Actionability**  
*Could you actually use this feedback to improve?*
- **Rating:** Select from dropdown (5-Excellent to 1-Poor)
- **Evidence:** Provide a specific example or observation  
  *Example: The agent suggested adding "This discovery matters because..." which gave me a concrete template to follow.*

**Criterion 5: Affective Quality**  
*Did the feedback motivate you to improve?*
- **Rating:** Select from dropdown (5-Excellent to 1-Poor)
- **Evidence:** Provide a specific example or observation  
  *Example: "The agent started with positive observations about my content accuracy before suggesting improvements, which felt encouraging rather than discouraging."*

**Overall Assessment**
- **Overall Rating:** Based on all criteria, how would you rate this agent overall? (5-Excellent to 1-Poor)
- **Overall Feedback:** Summarize your experience. What worked well? What should be improved?  
  *Example format:*
  - *What worked well: Specific quotes from my text helped me understand exactly what to improve*
  - *What should be improved: Could prioritize which issues to fix first*

### Submit:
- Review your responses
- Click **Submit new issue**
