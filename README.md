# Can AI Give Good Feedback? Build One and Find Out

**Amsterdam Tech Week Hackathon**  
June 17, 2026 | 18:00–21:00  
FeedbackFruits, Amsterdam

---

## The Challenge

Can you build an AI agent that gives **better feedback** than your peers?

Design, test, and refine an AI-powered feedback agent for science writing. Your agent helps students improve their summaries using the **Universal Science Writing Rubric**.
